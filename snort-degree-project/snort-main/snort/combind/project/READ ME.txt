🔎 Snort - A Full-Stack Search Engine

Snort is a lightweight full-stack search engine built using Python Flask, SQLite, HTML, and CSS. It features a secure login and registration system and allows users to perform real-time searches using public APIs like **Wikipedia** and **IMDb**.
